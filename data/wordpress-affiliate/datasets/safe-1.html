This screen provides access to your site’s referral history.

Reference: this refers to the order number (or similar) that created this referral.

Referral statuses:

  • Paid - this is a referral that has been paid to the affiliate
  • Unpaid - this is a referral that has been accepted but not yet paid to the affiliate
  • Pending - this is a referral that is waiting to be accepted
  • Rejected - this is a referral that has been rejected and will not be paid to the affiliate

Referrals can be searched in several different ways:

  • You can enter the referral’s ID number
  • You can enter the referral reference prefixed by “ref:”
  • You can enter the referral context prefixed by “context:”
  • You can enter the affiliate’s ID number prefixed by “affiliate:”
  • You can enter a URL campaign prefixed by “campaign:”

Referrals can be exported directly to a CSV file in order to make it easier for your own accounting needs and for you to payout your affiliates’s earnings.

The CSV file generated is structured properly for PayPal’s Mass Payment system so you can easily payout all of your affiliates at once via PayPal.
